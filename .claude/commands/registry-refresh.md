---
name: registry-refresh
description: Regenerate .claude/skill-registry.md by scanning all command files in .claude/commands/.
---

Regenerate the skill registry from the current set of command files.

## Steps

1. Glob all `.md` files in `.claude/commands/` except `registry-refresh.md` itself.

2. For each file, read it and extract from the YAML frontmatter:
   - `name` — the slash command name
   - `description` — one-line summary

3. Also read the first non-empty line of the file body (after the closing `---`) to use as the "purpose" line if the description alone is too short.

4. Write `.claude/skill-registry.md` with the format below. Replace the `Last updated` date with today's date.

5. Confirm: "Registry refreshed — N skills indexed." Do not list all skills in the response; the file is the output.

## Output format

```markdown
# Skill Registry

_Auto-generated by /registry-refresh — do not edit manually._  
_Last updated: YYYY-MM-DD_

## Skills

| Command | Description | When to use |
|---|---|---|
| `/name` | description from frontmatter | first body line |
...
```

The "When to use" column should be a short phrase (5–10 words) derived from the body's opening sentence — not the full description.
